# Tampermonkey Userscripts

A collection of Tampermonkey userscripts organized by website.

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) in your browser
2. Create a new script, paste the contents of the desired `.user.js` file
3. Save and reload the target website

## Scripts

### [mydofus.de](https://mydofus.de)

| Script | Description |
|--------|-------------|
| [Territories Sorter](mydofus.de/mydofus-sort-territories.user.js) | Sorts territories alphabetically within each area on the `/territories` page. Detects the user's language for locale-aware sorting. Works on initial load and SPA navigations. |
